# Weather_bot.py
Структура проекта 
weather_bot/
├── bot.py                  # основной код бота
├── Dockerfile              # инструкция для сборки образа
├── docker-compose.yml      # описание сервисов
├── requirements.txt        # зависимости Python
└── .env                    # секретные переменные окружения
Для старта:
docker compose -d
Функционал:
Выводит информацию о погоде в определенном городе, можно подписаться на рассылку в 8:00 о погоде в своем городе.
Можно изменить время.
